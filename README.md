# Java gyakorlás 

Rövid ismertető a projekt(ek)ről: 
* A feladatok [IntelliJ IDEA][IDEA] IDE segítségével írodtak. 
* A feladatok többnyire ezekbe a _csoportokba_ oszthatóak be:
